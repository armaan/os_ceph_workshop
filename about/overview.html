About This Course

This session will provide a hands on learning experience of deploying OpenStack and Ceph with Ansible. We will use openstack-ansible for deploying OpenStack and ceph-ansible for deploying Ceph.

Objectives

At the end of the module you will know

  • How to install OpenStack with openstack-ansible;
  • How to install Ceph with ceph-ansible;
  • How to integrate OpenStack with Ceph;
  • How to enable Swift API compatible access to your Ceph with radosgw;
  • How to destroy and rebuild service specific containers;
  • How to manage Galera cluster failure;
  • How to manage RabbitMQ cluster partitions.
